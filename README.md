**Назначение:**
Объединяет все компоненты в один исполняемый файл. Позволяет запустить полный цикл работы программы одной командой без необходимости открывать несколько файлов.

**Что делает:**
1. Создает базу данных `photomaterials_orders.db`
2. Создает все 6 таблиц с правильными связями
3. Заполняет таблицы тестовыми данными
4. Демонстрирует работу всех функций системы
5. Показывает примеры создания, просмотра и редактирования заявок

**Запуск:**
`py run.py`

#### `SVG файлы`
**Визуализация структуры базы данных (ER-диаграмма)**
Содержит саму ER-диаграмму и вторую, с объяснением значений

**Назначение:**
Графическое представление структуры базы данных, показывающее все таблицы, их поля, типы данных, ключи и связи между таблицами.

**Что отображает:**
**Таблицы:**
1. **PARTNERS** - информация о партнерах-заказчиках
2. **MANUFACTURERS** - производители фотоматериалов
3. **PRODUCT_TYPES** - категории товаров
4. **PRODUCTS** - список товаров
5. **ORDERS** - заявки партнеров
6. **ORDER_ITEMS** - позиции в заявках
